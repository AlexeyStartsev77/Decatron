На данный момент собран один протатип по старой схеме и все исследования идут на нем.
После вдумчивого чтения множества различных форумов и общения с умными людьми, а так же чтения бреда ИИ - родилась "новая" схема, но прототип по ней пока не собран.

Что умеет действующий прототип по старой схеме:
1. статические эффекты без гашения заряда.
2. динамические эффекты без гашения заряда.

Что прототип не умеет:
1. адресно зажигать К0 (при открытии его ключа - заряд зажигается где угодно)
