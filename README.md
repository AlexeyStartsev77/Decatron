На данный момент собран один протатип по старой схеме и все исследования идут на нем.
После вдумчивого чтения множества различных форумов и общения с умными людьми, а так же чтения бреда ИИ - родилась "новая" схема, но прототип по ней пока не собран.

Что умеет действующий прототип по старой схеме:
1. статические эффекты
2. динамические эффекты
3. адресно зажигать К0


Реализованы эффекты:
1. Крутилка по часовой стрелке
2. Крутилка против часовой стрелки
3. "Улыбка" (дуга заданной ширины внизу декатрона)
4. Динамическая "улыбка"
5. Часы (заполнение, а затем убывание - по часовой стрелке)
6. Маятник (**в разработке)
7. Мигалка K0

для некоторых эффектов приложены видео

В планах:
1. добавить отладочный/информационный дисплей (возможно oled 128х64)
2. добавить управление энкодером (0-ой уровень - переличтывание эффектов, 1-уровень (одно нажание для вход/выход) - изменение скорости, 2-уровень (двойное нажатия для вход/выход) - прочие настройки (возможно, изменение направления эффекта))
